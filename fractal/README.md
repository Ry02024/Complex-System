# **コッホ曲線 - Koch Curve Animation**
コッホ曲線は、自己相似性を持つフラクタル構造の一つです。本プロジェクトでは、Pythonを用いてコッホ曲線を再帰的に生成し、アニメーションとして可視化する方法を学べます。

---

## 📌 **内容**
- フラクタルの基本理論
- Pythonによるコッホ曲線のシミュレーション実装
- 可視化の方法（Matplotlib + FuncAnimation）
- 応用例（フラクタル圧縮、形状生成、自然パターン）
